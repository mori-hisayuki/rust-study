## Rustの学習
https://doc.rust-jp.rs/rust-by-example-ja/index.html のページを参考にRustの学習を行う

## 環境構築

### [Taskfile](https://taskfile.dev/)のインストール
#### macOS
Homebrewを使ってインストールします。
`Taskfile`のインストール
```bash
brew install go-task
```

### セットアップ
以下のコマンドを実行する。
```bash
task setup
```
